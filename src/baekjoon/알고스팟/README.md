## 오답노트
처음에는 두 번째 테케가 안 돌아가서 무한루프에 걸린줄 알았는데, N, M의 순서가 잘못되어있었다.
당연히 N, M순서로 입력이 주어질줄 알았는데 M, N의 순서로 입력이 주어졌다.
비슷한 상황이 생기면 입력 순서 이슈도 체크해보자.
