## 오답노트
다익스트라 문제에서 경유 노드를 체크할 때는 실제로 해당 노드를 체크하지 않고도 목적지에 같은 비용을 감수하고 가는 경우가 존재할 수 있다.
따라서 해당 노드까지 간 거리와 그 노드에서 목적지까지 간 거리의 합이 출발지점에서 목적지까지 한 번에 간 거리와 같은지를 확인해야한다.
