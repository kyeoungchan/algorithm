### 어려웠던 점
- 주어진 테케는 다 맞는데 자꾸 제출할 때마다 런타임 에러가 발생하였다.
  - 결국 마지막 7번 반례 덕분에 해결할 수 있었다.
  - 문제에서 주어진 r과 c는 처음에 3x3 크기인 A 행렬에 접근할 수 있다는 보장이 없었다.
  - 따라서 r과 c를 비교할 때는 A 크기 범위에 있는지 확인부터 하는 것을 넣자 해결되었다.
